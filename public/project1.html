Built and evaluated four ML models using Python (scikit-learn):

  • Gradient Boosting Classifier
  • Linear Regression (for multi-level targets)

Feature importance highlighted immune evasion capacity, systemic tissue involvement, and indirect transmission routes as critical drivers of severity. Evaluation used AUC-ROC, precision, recall, and F1 scores.
